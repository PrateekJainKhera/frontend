# Install Missing Packages

Run these commands to install missing Radix UI dependencies:

```bash
cd c:\Users\prate\OneDrive\Desktop\multihitech\frontend

# Install missing Radix UI components
npm install @radix-ui/react-separator @radix-ui/react-dropdown-menu @radix-ui/react-tabs @radix-ui/react-alert-dialog
```

Then start the dev server:
```bash
npm run dev
```
